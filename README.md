# Neil Joseph — Personal Portfolio

> Senior Software Engineer · Product Owner · Scrum Master

A personal portfolio website for **Neil Joseph** — a Senior Software Engineer with Product Owner and Scrum Master experience, and 7 years in the IT industry. The site presents Neil's professional identity at the intersection of engineering and product, with a clean, light techno aesthetic.

---

## Overview

This is a single-file static website (`index.html`) built with plain HTML, CSS, and JavaScript — no frameworks, no dependencies, no build step required. It is designed to be hosted directly via GitHub Pages.

The portfolio covers five sections:

- **Hero** — Introduction, role summary, availability status, and a terminal-style profile card
- **Tech Stack & Skills** — Languages, frameworks, cloud tools, role expertise matrix, and proficiency bars
- **Work Experience** — A chronological timeline of professional roles with responsibilities and outcomes
- **Certifications** — A focused showcase of the Certified Scrum Master (CSM) credential
- **Contact** — Location, email, LinkedIn, and GitHub details

---

## Purpose

This portfolio serves as Neil's primary professional web presence — replacing a static CV with a living, accessible, and visually distinctive page that reflects both his technical depth and his product mindset.

It is intentionally minimal in its dependencies and infrastructure so that it remains easy to update, own, and host independently.
